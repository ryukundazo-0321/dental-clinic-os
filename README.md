# 🦷 DENTAL CLINIC OS

**AIありき設計の歯科経営OS**

人ありきの組織を、AIありきの組織構造へ。  
すべてを一元管理し、人の介在を最小限にした歯科業界の新しい産業OSです。

## 技術スタック

- **フロントエンド**: Next.js 14 (React) + Tailwind CSS
- **データベース**: Supabase (PostgreSQL)
- **ホスティング**: Vercel
- **AI**: Claude API, Whisper API

## 業務フロー（4フェーズ）

| Phase | 内容 | 状態 |
|-------|------|------|
| Phase 1 | 予約〜問診（来院前） | 🔨 開発中 |
| Phase 2 | 受付〜案内（来院時） | ⏳ 予定 |
| Phase 3 | 診察・処置（診療中） | ⏳ 予定 |
| Phase 4 | 会計・精算（診療後） | ⏳ 予定 |

## セットアップ

```bash
# 依存パッケージのインストール
npm install

# 環境変数の設定
cp .env.example .env.local
# .env.local にSupabaseの接続情報を入力

# 開発サーバー起動
npm run dev
```

## 設計原則

- 人が介在するのは「診察・処置」のみ
- 各ステップの完了が次のステップのトリガー（イベント駆動設計）
- 全データは一元管理、二重入力は一切発生しない
- 予約から会計まで、データが自動で流れる仕組み
